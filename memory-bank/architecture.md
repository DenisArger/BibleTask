# Architecture

Основные модули:
- `index.html` — контейнеры интерфейса и подключение скриптов.
- `app.js` — бизнес‑логика загрузки данных и рендеринга.
- `arrayBooks.js` — справочник книг и названий.
- `config.js` — маппинг для перекрёстных ссылок и имён.

Поток данных:
1. Запрос списка книг через API.
2. Пользователь выбирает книгу и главу.
3. Загружается текст выбранной главы.
4. Текст распределяется по двум колонкам.
5. Загружаются и выводятся перекрёстные ссылки.
