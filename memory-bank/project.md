# Project Summary

BiblePages — статическая веб‑страница для чтения библейского текста.

Ключевые функции:
- выбор книги и главы;
- загрузка текста главы через API biblesupersearch;
- двухколоночный вывод текста;
- блок перекрёстных ссылок по стихам.

Данные и справочники:
- список книг и их сокращения хранится в `arrayBooks.js` и `config.js`;
- перекрёстные ссылки берутся из внешних JSON‑файлов.
