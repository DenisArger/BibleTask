# Memory Bank

Этот каталог хранит краткую «память проекта»: что это за проект, как он устроен, какие решения приняты и какие задачи запланированы. Файлы обновляются по мере изменений в проекте.

Состав:
- `project.md` — краткое описание и ключевые функции.
- `architecture.md` — структура и поток данных.
- `decisions.md` — журнал решений и компромиссов.
- `todos.md` — список ближайших задач.
